# Kaizen Linux v0.1.0-alpha

First installable alpha release of Kaizen Linux.

## Confirmed working

- UEFI ISO boot
- Hyprland live session
- Calamares installer launch
- Successful UEFI installation
- Installed system boots into Hyprland
- Installed system identifies as Kaizen Linux
- Default Kaizen wallpaper included

## Known limitations

- Secure Boot is not supported yet
- Use UEFI without Secure Boot for testing
- Wallpaper selector is not finished
- Quickshell/AetherDeck is not packaged yet
- Some Fedora internals may still appear in low-level tools
