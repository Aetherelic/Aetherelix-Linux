Welcome to Kaizen Linux

Kaizen Linux is a Fedora-based Hyprland desktop built to be polished, fast, and beginner-friendly.

Essential keybinds

Super + Enter Open terminal

Alt + Space Open app launcher

Super + Q Close active window

Super + R Reload Hyprland

Super + F Fullscreen

Super + 1-5 Switch workspace

Super + Shift + 1-5 Move window to workspace

Updating Kaizen

To pull the latest Kaizen desktop fixes, run:

kaizen-update

Kaizen profiles

Optional profiles let you add more software when you need it.

kaizen-install-profile gaming Install gaming tools

kaizen-install-profile productivity Install productivity tools

kaizen-install-profile all Install both profiles
