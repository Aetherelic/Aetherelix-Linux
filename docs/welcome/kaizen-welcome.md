# Welcome to Kaizen Linux

Kaizen Linux is a Fedora-based Hyprland desktop built for a polished, beginner-friendly Linux experience.

## Essential keybinds

- Super + Enter: Open terminal
- Alt + Space: Open app launcher
- Super + Q: Close active window
- Super + R: Reload Hyprland
- Super + F: Fullscreen
- Super + 1-5: Switch workspace
- Super + Shift + 1-5: Move window to workspace

## Getting started

Use the app launcher to open your apps.

Kaizen includes a preconfigured Hyprland desktop, Waybar, Rofi, Kitty, Fastfetch, wallpaper support, fonts, icons, and essential desktop tools.
