# Aetherelix Linux First Boot

Welcome to the early Aetherelix Linux Fedora Hyprland test image.

## Default VM-safe keybinds

- Ctrl + Alt + T: open terminal
- Alt + Space: open launcher
- Alt + Q: close window
- Alt + M: exit Hyprland
- Alt + 1-5: switch workspaces
- Alt + Shift + 1-5: move window to workspace
